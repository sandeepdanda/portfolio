# portfolio

Source for my personal website.

Built with Astro, Tailwind, and MDX. Deployed to Cloudflare Pages (TBD).

## Local dev

```bash
npm install
npm run dev
```

Open http://localhost:4321

## Build

```bash
npm run build
```

Output goes to `dist/`.

## Structure

- `src/pages/` - routes
- `src/content/` - case studies and project entries (MDX)
- `src/components/` - reusable Astro components
- `src/layouts/` - page shell
- `src/styles/` - global CSS and design tokens
- `docs/` - design notes and build plan
